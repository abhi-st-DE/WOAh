# Learning Agents

A growing collection of hands-on projects for learning how AI agents are designed, built, and improved over time.

## Current Progress

✅ **Project 001 completed:** a quality weather agent built from scratch, with robust weather retrieval using:
- **wttr.in** as the primary source
- **OpenWeather API** as a fallback source

This setup demonstrates practical agent engineering: tool use, fallback reliability, and user-facing responses.

## Repository Structure

- `project_001_Agent_from_scratch/` — first complete agent project (weather agent)

## Why this repository exists

This repository is focused on incremental, project-by-project learning:
- Build real agents end-to-end
- Understand tool-calling workflows
- Improve reliability with layered data sources
- Keep experiments organized and reproducible

## Next Direction

Additional agent projects will be added as separate project folders, each focused on a distinct capability and learning objective.
